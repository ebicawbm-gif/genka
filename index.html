クイック操作

よく使う系だけ右に置いとく。
邪魔な単品ボトルは、エイリアス空白で放置＝レシピ候補に出ない。

原価マスター（Base×Price → 1g/1ml/1個の原価）

Base（固定）を整えるとレシピ入力が爆速。Price（棚卸）はCSVで更新される想定。
エイリアス空白＝レシピ候補に出さない（単品ボトル等の除外に使う）。
Ver0.6：入力中にスクロールが飛ばない（行だけ表示更新）。
商品名（key） エイリアス（Base） よみ（Base） 単価(円)（Price） 価格単位（Price） 規格/入数（Price） 基準量（Base） 原価（Base×Price） メモ（Base） 操作
OK = 基準量(Base) > 0 かつ 単価(Price) > 0。
商品によって「本単価」なのに規格に24/C/Sが付くことがあるので、基準量(Base)を正に寄せる運用。

レシピ原価（材料×使用量 → 自動原価）

材料は エイリアス / よみ(かな) / 商品名 で検索して候補から選ぶ。
使用量プリセット を登録して、行の横ボタンで一発入力できる（例：リキュール30ml / ビール300ml）。
ショートカット：Ctrl + S で保存
入力すると「1杯(1人前)あたり原価」も表示。
